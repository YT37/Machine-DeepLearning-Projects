# StockPredictor
This project Predicts future stock prices based on their Features using LSTM.

# Requirements
 - Python 3.6 or up
 - TensorFlow
 - SciKitLearn
 - NumPy
 - Pandas
 - MatplotLib
 
 # Instructions To Use
 - **Windows, Mac and Linux**
   ``` 
   pip install -r requirements.txt
   ```
 - **Windows**
   ```
   python StockPredictor.py
   ```
 - **Mac or Linux**
   ```
   python3 StockPredictor.py
   ```
